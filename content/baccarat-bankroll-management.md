# 百家樂資金管理實戰指南：停損停利設定、注碼分配與凱利公式應用

百家樂最常見的輸法不是技術差，而是資金管理崩盤。

你一定聽過這種故事：有人帶了 10 萬上桌，前半小時贏了 3 萬，結果不到兩小時全部輸光。問題不在他看路不準或運氣不好，問題在他根本沒有「什麼時候該走」和「每手該壓多少」的計畫。

這篇文章不會教你什麼必勝打法——那種東西不存在。我們要談的是**資金管理**：用數學設定停損停利、用公式決定注碼大小、用數據輔助出手時機。這些不會讓你變成莊家，但能讓你不再當韭菜。

## 為什麼資金管理比「攻略」重要？

**資金管理是你在百家樂牌桌上唯一能完全控制的事情。**

你控制不了發牌結果，控制不了牌靴剩餘組合，但你能控制兩件事：**下不下注**，以及**下多少**。而這兩個決定，就是資金管理的全部核心。

### 攻略的致命缺陷

網路上充斥各種百家樂「攻略」——看路法、纜法、追龍法。這些方法的共同問題是：**它們無法改變期望值**。

百家樂的[莊家優勢](/blog/baccarat-house-edge-explained)是固定的數學結構。8 副牌起手時，押莊的期望值約 -1.06%，押閒約 -1.24%。不管你用什麼路子、什麼纜法，只要你在標準起手狀態下注，每一手的期望值就是負的。

那資金管理有什麼不同？它不改變期望值，但它做兩件更實際的事：

1. **控制破產風險**——確保你不會因為一段下風期就把本金全部燒完
2. **最大化正 EV 時機的收益**——當數學站在你這邊時，用合理的注碼放大優勢

如果你還不清楚什麼是期望值，建議先讀這篇：[百家樂 EV 期望值完整解析](/blog/baccarat-ev-expected-value-explained)。

## 停損停利怎麼設定才科學？

**停損和停利不是「感覺差不多就走」，而是出門前就寫死的數字。**

很多人的停損停利是這樣的：「輸太多就走」、「贏夠了就收」。這種模糊的標準在情緒激動的牌桌上毫無約束力。你需要的是一組具體的數字，寫在紙上，進場前就決定好。

### 停損設定：保護本金的紅線

停損的目的很明確——**限制單次 session 的最大虧損，確保你還有下一次上桌的機會。**

常見的停損比例建議：

| 玩家類型 | 單次 Session 停損 | 說明 |
|---------|-------------------|------|
| 保守型 | 總資金的 10% | 10 次連續最大虧損才會破產 |
| 穩健型 | 總資金的 15-20% | 多數職業玩家的選擇 |
| 激進型 | 總資金的 30% | 波動大，破產風險顯著提高 |

**具體範例：**

假設你的百家樂專用資金（bankroll）是 50,000 元。

- 保守型停損：50,000 × 10% = **5,000 元**。這場最多輸 5,000 就走。
- 穩健型停損：50,000 × 20% = **10,000 元**。最多輸 10,000。

重點不是哪個「對」，而是你**選了就不能改**。輸到停損線，站起來走人，沒有任何例外。

### 停利設定：鎖住已到手的利潤

停利比停損更難執行，因為人在贏錢的時候最不想離開。

兩種常見的停利策略：

**固定停利**：設定一個獲利目標，到了就走。
- 例：帶 10,000 上桌，目標獲利 3,000（30%），到了就收。

**回撤停利**：不設上限，但從最高點回撤一定比例就走。
- 例：你贏到 8,000 利潤，設定回撤 30%。如果利潤從 8,000 掉到 5,600（回撤 2,400），立刻走人。

回撤停利的好處是：如果你正在「順風」，不會過早離場；但一旦風向轉了，也不會把利潤全部吐回去。

**推薦組合：固定停損 + 回撤停利。**

停損寫死一個數字，不可變。停利用回撤機制，讓贏的時候能多贏，但設一條底線保住部分利潤。

## 注碼分配策略：每手該壓多少？

**注碼大小決定了你的波動範圍和存活時間。**

這是資金管理最核心也最被忽略的部分。多數人壓注靠感覺——贏了就加碼，輸了也加碼想翻本。這兩種行為都是通往破產的快車道。

以下比較三種主流的注碼分配策略。

### 策略一：固定注碼法

每一手都壓相同金額，不管輸贏。

- **計算方式**：bankroll ÷ 手數 = 單手注碼
- **範例**：bankroll 50,000 元，計畫打 200 手，每手壓 250 元
- **優點**：簡單、波動可控、不會情緒化加碼
- **缺點**：贏的時候沒有加速，效率偏低

適合新手和嚴格自律的玩家。如果你不確定該用什麼策略，固定注碼永遠是最安全的起點。

### 策略二：比例注碼法

每手壓當前 bankroll 的固定百分比。

- **計算方式**：當前 bankroll × 固定比例 = 單手注碼
- **範例**：bankroll 50,000 元，比例 2%。第一手壓 1,000。如果 bankroll 漲到 55,000，下一手壓 1,100；如果跌到 45,000，下一手壓 900。
- **優點**：自動調節——贏多壓多，輸多壓少，理論上不會歸零
- **缺點**：連輸時注碼越來越小，回本速度慢

### 策略三：凱利公式（Kelly Criterion）

根據你的優勢大小動態調整注碼——優勢越大壓越多，沒優勢就不壓。

這是三種策略中數學上最優的，也是下一節的重點。

### 三種策略模擬比較

以下模擬假設初始 bankroll 50,000 元，進行 500 手，平均每手期望值從 -1.2% 到 +0.5% 不等（模擬有正 EV 出手時機的場景）：

| 策略 | 單手注碼邏輯 | 500 手後平均 bankroll | 破產機率 | 最大回撤 |
|------|-------------|----------------------|---------|---------|
| 固定注碼（500 元/手） | 固定 500 | 49,200 | 2.1% | -12,500 |
| 比例注碼（1%） | bankroll × 1% | 49,450 | 0.3% | -9,800 |
| 凱利公式（僅正 EV 下注） | 依優勢動態計算 | 51,800 | 0.1% | -7,200 |
| 無策略（感覺下注） | 500-5,000 隨機 | 41,500 | 18.7% | -32,000 |

數據很直觀：**無策略的「感覺下注」破產風險是凱利公式的 187 倍。**

## 凱利公式在百家樂怎麼用？

**凱利公式告訴你：根據你的優勢大小，最佳注碼佔 bankroll 的百分之幾。**

### 凱利公式的原始公式

```
f* = (bp - q) / b
```

其中：
- **f*** = 最佳下注比例（佔 bankroll 的百分比）
- **b** = 賠率（贏了賺多少倍）
- **p** = 勝率
- **q** = 敗率（q = 1 - p）

### 百家樂的凱利計算範例

**範例一：標準起手狀態押莊**

在 8 副牌起手狀態下押莊：
- 莊勝率 p = 45.86%（扣除和局後）
- 莊賠率 b = 0.95（贏了賺 0.95 倍，因為抽 5% 佣金）
- 敗率 q = 54.14%

```
f* = (0.95 × 0.4586 - 0.5414) / 0.95
f* = (0.4357 - 0.5414) / 0.95
f* = -0.1057 / 0.95
f* = -0.1113 = -11.13%
```

**凱利值是負的。** 這代表什麼？代表數學告訴你：**不要下注。** 標準起手狀態下，百家樂是負 EV 遊戲，凱利公式給出的最佳注碼是零。

這就是資金管理最殘酷也最誠實的結論——大部分時候，最佳策略是不下注。

**範例二：正 EV 出手時機**

隨著牌靴發牌進行，剩餘牌組成會改變，某些特定時刻可能出現正 EV。假設某一手的數據：
- 押莊勝率 p = 51.2%（剩餘牌組合有利於莊）
- 賠率 b = 0.95
- 敗率 q = 48.8%

```
f* = (0.95 × 0.512 - 0.488) / 0.95
f* = (0.4864 - 0.488) / 0.95
f* = -0.0016 / 0.95
f* = -0.0017
```

優勢太小，凱利值仍然接近零。再看一個更明顯的例子：

**範例三：顯著正 EV 時機**

假設剩餘牌組合極端有利：
- 押閒勝率 p = 53.5%
- 賠率 b = 1.0（閒不抽佣）
- 敗率 q = 46.5%

```
f* = (1.0 × 0.535 - 0.465) / 1.0
f* = 0.07 = 7%
```

凱利公式建議你把 bankroll 的 7% 壓在這一手。如果你的 bankroll 是 50,000 元，單手注碼就是 3,500 元。

### 半凱利：實務上的安全做法

完整凱利公式的波動性很大。職業玩家通常使用**半凱利（Half Kelly）**——把計算結果除以 2。

上面的範例用半凱利：f* = 7% ÷ 2 = **3.5%**，單手壓 1,750 元。

半凱利犧牲了一點長期收益，但大幅降低了短期波動和破產風險。對大多數人來說，這是更務實的選擇。

### 凱利公式的關鍵前提

凱利公式要好用，有一個絕對前提：**你必須知道準確的勝率和賠率。**

在百家樂中，這意味著你需要即時計算剩餘牌組合的勝率——這不是人腦能做到的事。這正是數據工具存在的意義，後面會詳細說明。

## EV 數據 + 資金管理：真正的整合打法

**最好的資金管理 = 只在正 EV 時下注 + 用凱利公式決定注碼。**

到這裡，我們可以把前面所有概念整合成一套完整的資金管理系統。

### 系統的三層架構

**第一層：是否下注（EV 篩選）**

不是每一手都要下注。只有當期望值為正時，才值得出手。其餘時間，你的最佳策略就是坐著看。

這需要即時計算剩餘牌面的 EV 值——[窮舉法](/blog/baccarat-ev-expected-value-explained)是目前最精確的做法，它計算所有可能的發牌組合，給出真正的數學期望值。

**第二層：壓多少（凱利注碼）**

一旦確認正 EV，用凱利公式（或半凱利）根據優勢大小決定注碼。優勢大就多壓，優勢小就少壓，精準分配每一塊錢的風險。

**第三層：何時離場（停損停利）**

即使有正 EV 出手時機，短期波動仍然存在。停損停利是你的安全網，確保單次 session 的虧損不會傷及整體 bankroll。

## 常見的資金管理錯誤（你中了幾個？）

**大多數人的資金管理失敗不是因為不懂，而是因為做不到。**

### 錯誤一：沒有獨立的 bankroll

把百家樂的資金和生活費混在一起，是最常見也最致命的錯誤。你必須有一筆專門用於百家樂的資金，這筆錢即使全部輸光也不影響生活。

### 錯誤二：輸了就加碼追回

典型的「倍壓追損」——輸 1,000 就壓 2,000，再輸就壓 4,000。馬丁格爾策略（Martingale）在數學上已經被證明是失敗的：它不改變期望值，只是把「小虧」的機率轉換成「爆倉」的機率。詳細分析見[注碼系統數學分析](/blog/baccarat-betting-systems-math-analysis)。

### 錯誤三：贏了就覺得自己無敵

連贏幾手之後開始大幅加碼，覺得自己「手感正熱」。但百家樂每一手都是獨立事件（在不考慮剩餘牌組成的前提下），上一手贏不代表下一手會贏。

### 錯誤四：不設停損或到了不執行

設了停損 5,000 但輸到 5,000 的時候告訴自己「再打幾手回本」——這比不設停損更糟，因為你在欺騙自己。

### 錯誤五：在負 EV 狀態下瘋狂下注

百家樂大部分時間都是負 EV。如果你每一手都下注，你就是在用最笨的方式把錢送給莊家。學會等待，只在數學有利的時候出手，是資金管理最高層次的紀律。

## 常見問題 FAQ

### Q1：百家樂資金管理真的有用嗎？不是一樣會輸？

資金管理不會把負 EV 變成正 EV，但它能做到兩件事：(1) 大幅降低你單次爆倉的風險；(2) 讓你在出現正 EV 機會時還有足夠的本金可以出手。把它想成保險——不能幫你賺錢，但能確保你不會一夜之間從遊戲中被淘汰。

### Q2：百家樂停損設多少比較合理？

建議單次 session 停損設在總 bankroll 的 10%-20%。具體數字取決於你的風險承受度和上桌頻率。如果你每天都打，停損應該設低一點（10%）；如果一個月打一兩次，可以稍微寬一些（20%）。核心原則是：連續輸 5-10 次最大虧損後，你還有足夠的 bankroll 繼續玩。

### Q3：凱利公式在百家樂實用嗎？一般人算得出來嗎？

凱利公式本身很簡單，但前提是你需要知道精確的勝率。百家樂的勝率會隨剩餘牌組成變化，人工計算幾乎不可能。這就是為什麼需要工具輔助——像百家之眼這樣的系統能即時算出每一手的精確期望值，你再套入凱利公式就能得到最佳注碼。

### Q4：比例注碼和凱利公式有什麼差別？

比例注碼是每手壓 bankroll 的固定百分比（例如永遠壓 2%），不管這手的期望值是正還是負。凱利公式則是根據當手的優勢大小動態調整——優勢大就多壓，優勢小就少壓，沒優勢就不壓。凱利公式在數學上被證明是長期成長率最大化的策略，但前提是你有準確的勝率數據。

### Q5：為什麼說「不下注」也是一種資金管理？

因為在負 EV 的狀態下，每一次下注都是在做一筆虧本生意。選擇不下注，你的期望損失是零。這比任何注碼策略都好。資金管理最頂級的境界不是「怎麼壓」，而是「什麼時候不壓」。百家樂的正 EV 時機不常出現，學會等待就是最強的資金管理。
