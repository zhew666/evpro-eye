## 全局 SEO 規格

| 項目 | 規格 |
|------|------|
| 每篇字數 | 2,500~3,500 字（Google 偏好深度內容） |
| H1 | 每頁僅一個，含主關鍵字 |
| H2 | 3~6 個，含主詞或副詞變體 |
| H3 | 視需要，用於拆分 H2 段落 |
| Meta Title | 30 字以內，主關鍵字靠前 |
| Meta Description | 70~155 字，含主關鍵字 + CTA 暗示 |
| 圖片 ALT | 每篇至少規劃 2~3 張圖位（公式圖、流程圖、數據截圖） |
| 內部連結 | 每篇至少連向其他 2 篇文章 |
| Schema | Article + FAQ（每篇附 2~3 個 FAQ） |

## 法規注意事項

1. **絕對不能說**：保證贏、穩賺、必勝、破解（作為承諾用語時）
2. **可以說**：提高判斷依據、數學期望值分析、降低盲目下注、輔助決策工具
3. **免責聲明**：每篇文末固定加上「本文僅供數學研究與教學用途，不構成任何投注建議。百家樂為負期望值遊戲，請理性娛樂。」
4. **「破解」用法**：標題可用「破解」但文中定義為「破解迷思」「用數學破解錯誤觀念」，不是「破解賭場」
5. **不提供具體平台教學**：文章中不直接推薦特定娛樂城或下注策略
